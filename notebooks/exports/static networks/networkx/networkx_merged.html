Collaboration networks

Explanation:

Interval: The continuous time period observed at each snapshot of the network. E.g. a 7-day interval means that each t frame of the temporal network takes into account a 7-day period.

Step size: Time difference between the beginning of two consecutive network snapshots.

File-based and line-based network: Which algorithm was used to generate the network. File-based considers collaboration based on edits from two different authors on the same file. Line-based mode uses the git2net 'get_coauthorship_network' method, which considers collaboration based on edits in the same line or in the same block of code.
